The simple robot constructor.

The constructor, besides doing the normal constructor stuff, also calls the Initialize() C function where sensors can be set up immediately after the power is turned on.

void SimpleCRobot::StartCompetition ( )
virtual

Start a competition.

This code needs to track the order of the field starting to ensure that everything happens in the right order. Repeatedly run the correct method, either Autonomous or OperatorControl when the robot is enabled. After running the correct method, wait for some state to change, either the other mode starts or the robot is disabled. Then go back and wait for the robot to be enabled again.

Implements RobotBase.
