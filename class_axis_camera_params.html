Detailed Description

AxisCameraParams class.

This class handles parameter configuration the Axis 206 Ethernet Camera. It starts up a tasks with an independent connection to the camera that monitors for changes to parameters and updates the camera. It is only separate from AxisCamera to isolate the parameter code from the image streaming code.
